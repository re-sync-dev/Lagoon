Lagoon

Lagoon is a barebones typesafe framework that can be integrated anywhere.

Functions​

MakeService​

</>
Lagoon.MakeService(Module: T) → Service & T

Takes in a table for the 'Module' parameter and wraps the module with a Lagoon Service.

Usage:
MyService.lua

-- This is in a ModuleScript
local Lagoon = require(ReplicatedStorage.Packages.Lagoon)

local MyService = {}
MyService.IsService = true

function MyService.PrintFromServer()
	print("Hello from the server!")
end

return Lagoon.MakeService(MyService)

Load​

</>
Lagoon.Load(
List: Instance | {Instance},
ShouldTime: boolean?
) → (
number,
Dictionary<number>
)

Requires and initializes the provided modules. This function always returns the time in total it took to load the modules, however if ShouldTime is provded the second return value will have the time each module took to load.
